# Vim 编辑器速查

Vim 有三种主要工作模式：

| 模式 | 进入方式 | 说明 |
|------|----------|------|
| **普通模式（Normal）** | 启动 Vim 默认进入；按 `Esc` 返回 | 光标移动、复制、删除、查找等操作 |
| **插入模式（Insert）** | 在普通模式按 `i`、`a`、`o` 等键 | 输入和编辑文本内容 |
| **命令行模式（Command）** | 在普通模式按 `:` | 保存、退出、替换、设置等命令 |

---

## 一、普通模式

### 光标移动

#### 基本移动

| 按键 | 说明 |
|------|------|
| `h` / `←` | 光标向左移动一个字符 |
| `j` / `↓` | 光标向下移动一行 |
| `k` / `↑` | 光标向上移动一行 |
| `l` / `→` | 光标向右移动一个字符 |
| `n<Space>` | 向右移动 n 个字符（如 `20<Space>`） |
| `n<Enter>` | 向下移动 n 行（如 `5<Enter>`） |

#### 行内移动

| 按键 | 说明 |
|------|------|
| `0` | 移动到本行第一个字符 |
| `$` | 移动到本行最后一个字符 |
| `+` | 移动到下一行第一个非空格字符 |
| `-` | 移动到上一行第一个非空格字符 |

#### 页面跳转

| 按键 | 说明 |
|------|------|
| `Ctrl + F` | 向下翻一页 |
| `Ctrl + B` | 向上翻一页 |
| `Ctrl + D` | 向下翻半页 |
| `Ctrl + U` | 向上翻半页 |
| `H` | 移动到屏幕顶部第一行 |
| `M` | 移动到屏幕中间行 |
| `L` | 移动到屏幕底部最后一行 |

#### 文件跳转

| 按键 | 说明 |
|------|------|
| `G` | 移动到文件最后一行 |
| `gg` | 移动到文件第一行 |
| `nG` | 移动到第 n 行（如 `20G`，配合 `:set nu` 使用） |

---

### 查找

| 按键 | 说明 |
|------|------|
| `/word` | 向后（下）查找字符串 `word` |
| `?word` | 向前（上）查找字符串 `word` |
| `n` | 查找下一个匹配项 |
| `N` | 查找上一个匹配项 |

---

### 删除

| 按键 | 说明 |
|------|------|
| `x` | 向后删除光标所在字符 |
| `X` | 向前删除光标前一个字符 |
| `nx` | 向后删除 n 个字符（如 `3x`） |
| `dd` | 删除光标所在整行 |
| `ndd` | 删除光标所在行及向下 n 行（如 `5dd`） |
| `d1G` | 删除从光标所在行到文件第一行的所有内容 |
| `dG` | 删除从光标所在行到文件最后一行的所有内容 |

---

### 复制与粘贴

| 按键 | 说明 |
|------|------|
| `yy` | 复制光标所在行 |
| `nyy` | 复制光标所在行及向下 n 行（如 `3yy`） |
| `y1G` | 复制从光标所在行到文件第一行 |
| `yG` | 复制从光标所在行到文件最后一行 |
| `p` | 将复制内容粘贴到光标的**下一行** |
| `P` | 将复制内容粘贴到光标的**上一行** |

---

### 其他普通模式操作

| 按键 | 说明 |
|------|------|
| `u` | 撤销上一步操作（undo） |
| `Ctrl + R` | 重做（redo） |
| `J` | 将当前行与下一行合并为一行 |
| `.` | 重复执行上一次操作 |

---

## 二、插入模式

在普通模式下按以下键进入插入模式，按 `Esc` 返回普通模式。

| 按键 | 说明 |
|------|------|
| `i` | 在光标**当前位置**前插入 |
| `I` | 在当前行**行首**插入 |
| `a` | 在光标**当前位置**后插入 |
| `A` | 在当前行**行尾**插入 |
| `o` | 在当前行**下方**插入新行 |
| `O` | 在当前行**上方**插入新行 |
| `r` | 替换光标所在的单个字符 |
| `R` | 持续替换光标所在字符，直到按 `Esc` |
| `Esc` | 退出插入模式，回到普通模式 |

---

## 三、命令行模式

在普通模式下按 `:` 进入命令行模式。

### 保存与退出

| 命令 | 说明 |
|------|------|
| `:w` | 保存文件 |
| `:w!` | 强制保存（只读文件） |
| `:q` | 退出 Vim（未修改时） |
| `:q!` | 强制退出，不保存修改 |
| `:wq` | 保存并退出 |
| `:wq!` | 强制保存并退出 |
| `:x` | 保存并退出（与 `:wq` 等价） |
| `ZZ` | 普通模式下快捷保存退出（与 `:wq` 等价） |

### 文件操作

| 命令 | 说明 |
|------|------|
| `:w <filename>` | 将当前内容另存为新文件 |
| `:r <filename>` | 将指定文件内容插入到光标所在行后 |
| `:n1,n2 w <filename>` | 将第 n1 到 n2 行的内容保存为新文件 |

### 查找与替换

| 命令 | 说明 |
|------|------|
| `:n1,n2s/word1/word2/g` | 在第 n1 到 n2 行间，将 `word1` 替换为 `word2` |
| `:1,$s/word1/word2/g` | 全文将 `word1` 替换为 `word2` |
| `:1,$s/word1/word2/gc` | 全文替换，每次替换前询问确认（confirm） |
| `:%s/word1/word2/g` | 全文替换（`%` 等同于 `1,$`） |

### 显示设置

| 命令 | 说明 |
|------|------|
| `:set nu` | 显示行号 |
| `:set nonu` | 隐藏行号 |
| `:set syntax=on` | 开启语法高亮 |
| `:syntax enable` | 启用语法高亮 |
| `:set tabstop=4` | 设置 Tab 宽度为 4 个空格 |

---

## 四、语法高亮

```vim
" 开启语法高亮
:syntax enable

" 为特定语言加载语法高亮（如 PHP）
:source $VIMRUNTIME/syntax/php.vim
```

## 五、常用操作速查

### 快速入门流程

```
打开文件  →  vim filename
进入编辑  →  按 i 进入插入模式
编辑内容  →  正常输入文字
保存退出  →  按 Esc，输入 :wq，回车
```

### 常用操作组合

```vim
" 打开文件
vim filename

" 跳到第 30 行
:30

" 全文搜索并替换
:%s/old/new/g

" 显示行号并跳转到指定行
:set nu
:50

" 删除第 5 到 10 行
:5,10d

" 撤销所有修改，强制退出
:q!
```
