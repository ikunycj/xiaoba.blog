## 🎬 动画效果

### 主要动画
1. **bounce** - 上下弹跳 (头像、标签)
2. **wave** - 挥手动画 (👋)
3. **float** - 飘浮动画 (装饰元素)
4. **twinkle** - 闪烁动画 (✨)
5. **rotate** - 旋转动画 (虚线环)
6. **gradientMove** - 渐变移动 (背景)
7. **bounce-paw** - 爪印弹跳 (🐾)

### 交互动画
- **悬停**: translateY + scale + rotate
- **点击**: active 缩放反馈
- **滚动**: 进度条实时更新
- **加载**: bounceIn 入场动画

## 💡 使用建议

### 保持风格一致性
1. **配色**: 优先使用预设的粉蓝黄三色系
2. **圆角**: 统一使用 16-32px
3. **间距**: 使用 0.5rem 的倍数
4. **动画**: 时长 0.3s-0.4s，ease-in-out

### 添加新内容时
```css
/* 示例: 新卡片 */
.new-card {
  border: 3px solid var(--xb-border);
  border-radius: 24px;
  background: var(--xb-gradient-pink);
  box-shadow: var(--xb-shadow);
  transition: all 0.4s ease;
}

.new-card:hover {
  transform: translateY(-8px) rotate(-1deg);
  box-shadow: var(--xb-shadow-lg);
}
```

## 📊 性能优化

✅ **已实现**:
- CSS 动画使用 transform (GPU 加速)
- 背景渐变使用 CSS 而非图片
- 组件按需加载
- 图片使用 border-radius (避免 mask)

⚠️ **注意事项**:
- 飘浮装饰限制在 6 个以内
- 动画时长不超过 4s
- 阴影效果适度使用
- 移动端减少动画复杂度
