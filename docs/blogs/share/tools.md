# 🛠️ 效率工具推荐

精选开发者必备的效率工具，提升开发体验和生产力。

## 💻 开发工具

### IDE & 编辑器
- **[Visual Studio Code](https://code.visualstudio.com/)** - 现代化的轻量级代码编辑器
- **[WebStorm](https://www.jetbrains.com/webstorm/)** - 强大的 JavaScript IDE
- **[Cursor](https://cursor.sh/)** - AI 原生编程编辑器

### AI 编程助手
- **[GitHub Copilot](https://github.com/features/copilot)** - GitHub 官方 AI 编程助手
- **[Codeium](https://codeium.com/)** - 免费的 AI 代码补全工具
- **[CodeFlicker](https://codeflicker.com/)** - AI 驱动的代码开发平台

## 🎨 设计工具

- **[Figma](https://www.figma.com/)** - 协作式界面设计工具
- **[Excalidraw](https://excalidraw.com/)** - 手绘风格的在线白板
- **[draw.io](https://app.diagrams.net/)** - 免费的流程图绘制工具

## 📝 笔记 & 文档

- **[Obsidian](https://obsidian.md/)** - 强大的本地知识管理工具
- **[Notion](https://www.notion.so/)** - 全能的协作文档平台
- **[VitePress](https://vitepress.dev/)** - 基于 Vite 的静态站点生成器

## 🚀 部署 & 托管

- **[Vercel](https://vercel.com/)** - 前端项目快速部署平台
- **[Netlify](https://www.netlify.com/)** - 静态网站托管服务
- **[Cloudflare Pages](https://pages.cloudflare.com/)** - 免费的全球 CDN 部署

## 🔧 命令行工具

- **[Oh My Zsh](https://ohmyz.sh/)** - 强大的 Zsh 配置管理框架
- **[fzf](https://github.com/junegunn/fzf)** - 模糊搜索命令行工具
- **[tldr](https://tldr.sh/)** - 简化的命令行帮助文档

## 🌐 浏览器扩展

- **[Wappalyzer](https://www.wappalyzer.com/)** - 识别网站技术栈
- **[JSON Viewer](https://jsonviewer.stack.hu/)** - JSON 格式化查看
- **[React DevTools](https://react.dev/learn/react-developer-tools)** - React 开发调试工具
