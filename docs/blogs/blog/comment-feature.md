# 评论功能已上线：欢迎留言

站点现在已经接入评论功能，欢迎你在文章页留下想法、补充和建议。

## 你可以在哪里评论

- 在具体文章页的底部可以看到评论区。
- `blog` 和 `share` 下的大多数文章页会默认展示评论区。
- 如果某篇文章作者显式关闭了评论（`comment: false`），则不会显示。

## 首页的评论流

首页已经增加了评论流背景效果：

- 最新评论会以半透明气泡的形式从页面下方向上漂浮。
- 气泡会持续循环更新，便于快速感知站点互动。
- 点击气泡中的“查看文章”可以跳转到原文页面。

## 留言建议

- 可以直接发表文字评论，支持常见的 Markdown 内容。
- 如需附带音频，建议贴可访问的音频链接（如 `mp3`、`wav`、`m4a`、`ogg`）。
- 欢迎优先反馈勘误、补充资料和实战经验，这些对后续读者更有帮助。

## 说明

评论数据由 Twikoo 服务提供与存储，站点会展示公开留言内容用于社区互动。
