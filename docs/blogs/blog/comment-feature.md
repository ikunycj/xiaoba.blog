# 评论功能已上线：欢迎留言

站点现在已经接入评论功能，欢迎你在文章页留下想法、补充和建议。

## 你可以在哪里评论

- 在具体文章页的底部可以看到评论区。
- `blog` 和 `share` 下的大多数文章页会默认展示评论区。
- 如果某篇文章作者显式关闭了评论（`comment: false`），则不会显示。
- 博客列表页右侧还提供了一个固定留言板，适合提站点级建议。

## 评论方式

- 当前评论系统使用 Giscus。
- 评论内容托管在 GitHub Discussions 中。
- 发表评论需要先登录 GitHub 账号。

## 留言建议

- 可以直接发表文字评论，支持常见的 Markdown 内容。
- 欢迎优先反馈勘误、补充资料和实战经验，这些对后续读者更有帮助。

## 说明

评论数据由 GitHub Discussions 托管，站点通过 Giscus 嵌入讨论区。
